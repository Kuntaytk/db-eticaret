# db-eticaret
# E-Ticaret Veritabanı

Bu proje, SQL Server kullanılarak geliştirilmiş bir e-ticaret veritabanı tasarımıdır.

## Projenin Amacı

Bu proje, bir e-ticaret sisteminde bulunabilecek temel veritabanı yapısını modellemek ve ilişkisel veritabanı tasarımı konusunda deneyim kazanmak amacıyla hazırlanmıştır.

## Kullanılan Teknolojiler

* SQL Server
* T-SQL

## Veritabanı Özellikleri

* Müşteri yönetimi
* Ürün yönetimi
* Kategori yönetimi
* Sipariş yönetimi
* Tablolar arası ilişkiler
* Birincil ve yabancı anahtar kullanımı (Primary Key / Foreign Key)

## Veritabanı Yapısı

Veritabanında aşağıdaki temel tablolar bulunmaktadır:

* Müşteriler
* Ürünler
* Kategoriler
* Siparişler
* Sipariş Detayları

## Bu Projede Kazanılan Beceriler

* İlişkisel veritabanı tasarımı
* Tablo oluşturma ve yönetimi
* Veri ilişkileri kurma
* SQL sorguları yazma
* T-SQL kullanımı
